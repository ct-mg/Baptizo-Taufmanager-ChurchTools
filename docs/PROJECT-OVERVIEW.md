# Project Overview

## Baptizo Taufmanager
A ChurchTools extension for managing baptism processes, serving as a subtle branding tool for "Baptizo".

## Features
- **Legacy Integration**: Compatible with older ChurchTools deployments via `deploy:legacy`.
- **Live Connection**: Connects to `https://baptizo.church.tools/`.
- **Dashboard**:
  - 4-List Design for process tracking.
  - Interactive Charts for statistics.
  - Event Management integration.
- **Admin Protection**: Restricted area for system settings.
