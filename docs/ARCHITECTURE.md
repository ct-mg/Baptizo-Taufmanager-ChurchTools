# Architecture

## Technology Stack
- **Framework**: Vue 3 (Composition API)
- **Build Tool**: Vite
- **Language**: TypeScript

## Key Components
- **Dashboard**: Central hub with a 4-list layout (Open, In Progress, Done, Archived) and KPI visualization.
- **Charts**: Visual representation of baptism metrics using functional charts components.
- **Event Management**: Handling of baptism events and related workflows.
- **ChurchTools Integration**: Uses `churchtools-client` for API communication.
